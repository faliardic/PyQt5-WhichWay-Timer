\# Which Way? - Productivity Timer 🚀



A modern, minimalist timer application developed with \*\*Python\*\* and \*\*PyQt5\*\*. It helps track focus sessions by monitoring "Right Way" and "Wrong Way" activities.



\## Features ✨

\* \*\*Frameless Design\*\*: Modern and clean user interface.

\* \*\*Always on Top\*\*: Pin the app to stay visible over other windows.

\* \*\*Draggable\*\*: Move the timer anywhere on your screen.

\* \*\*Editable\*\*: You can directly click and edit the timer values.



\## Technical Details 🛠️

\* Developed using \*\*Object-Oriented Programming (OOP)\*\* principles.

\* Custom event handling for window movement.
